# Plan: De-AI the CCNA Quiz UI

**Goal:** Nâng điểm UI từ 5.5/10 lên 7.5+/10 bằng cách loại bỏ các dấu hiệu AI-generated, không đụng CLI block.

---

## Scope

- **Giữ nguyên:** CLI block styling (`.codeBlock`, `splitIntoSegments`, `looksLikeCli`)
- **Không làm:** Thay font (Tiempos/Styrene cần license)
- **Không làm:** Redesign HomeScreen wizard (scope quá lớn)

---

## Step-by-step

### 1. SVG icon set (6 icons) — `src/icons/`
Tạo file `src/icons/index.jsx` với 6 inline SVG components:
- `IconCheck` — dùng thay `✓` trong explanation status + option correct
- `IconX` — dùng thay `✕` trong explanation status + option incorrect
- `IconMenu` — dùng thay `≡` trong navToggle
- `IconClose` — dùng thay `✕` trong drawerClose
- `IconCheckbox` — dùng thay `☑` trong multipleBanner
- `IconArrowLeft` — dùng thay `←` trong exitButton

**Files:** `src/icons/index.jsx`

---

### 2. Bỏ emoji, dùng text/icon thuần — `QuizScreen.jsx`
- `'Hoàn thành 🎉'` → `'Hoàn thành'` (hoặc thêm confetti CSS animation nhỏ)
- `'☑ Chọn N đáp án đúng'` → dùng `<IconCheckbox />` + text
- `'≡'` navToggle → `<IconMenu />`
- `'✕'` drawerClose → `<IconClose />`
- `'← Thoát'` → `<IconArrowLeft />` + `'Thoát'`

**Files:** `src/components/QuizScreen.jsx`

---

### 3. Bỏ emoji trong OptionCard — `OptionCard.jsx`
- `'✓'` icon correct → `<IconCheck />`
- `'✕'` icon incorrect → `<IconX />`
- Checkbox `'✓'` trong `.checkbox` → `<IconCheck />`

**Files:** `src/components/OptionCard.jsx`

---

### 4. Fix options về 1 cột — `QuizScreen.module.css`
- Xóa `.options { grid-template-columns: 1fr 1fr }` → đổi thành `1fr`
- Xóa `.optionsFull` (không còn cần thiết)
- Xóa `hasCliOption` logic trong JSX (không cần nữa)
- Xóa `@media (max-width: 480px) { .options { grid-template-columns: 1fr } }` (redundant)

**Files:** `src/components/QuizScreen.module.css`, `src/components/QuizScreen.jsx`

---

### 5. Fix shadow màu warm — `QuizScreen.module.css`, `OptionCard.module.css`
- Tìm tất cả `rgba(0,0,0,0.08)`, `rgba(0,0,0,0.04)` trong shadow → đổi sang `rgba(120, 80, 50, 0.06)` và `rgba(120, 80, 50, 0.03)`
- Tìm `box-shadow: 0 4px 12px rgba(217, 119, 87, 0.2)` → giữ nguyên (đây là intentional glow)

**Files:** `src/components/QuizScreen.module.css`, `src/components/OptionCard.module.css`, `src/index.css` (nếu có global shadow token)

---

### 6. Fix backdrop-filter vô nghĩa — `QuizScreen.module.css`
- `.actions` background đang là `var(--color-bg)` (opaque) + `backdrop-filter: blur(8px)` → blur không có tác dụng
- Đổi background thành `rgba(250, 249, 245, 0.92)` để blur có nghĩa

**Files:** `src/components/QuizScreen.module.css`

---

### 7. Fix viBlock animation — `OptionCard.module.css`
- `.viBlock` dùng `max-height: 0 → max-height: none` không transition được
- Đổi sang cùng pattern với `.translateContent`: `grid-template-rows: 0fr → 1fr`
- Cần wrap content trong inner div để pattern hoạt động

**Files:** `src/components/OptionCard.jsx`, `src/components/OptionCard.module.css`

---

### 8. Fix hardcoded màu bypass token — tất cả files
Tìm và thay:
- `#1f5d33` (finishButton hover) → `color-mix(in srgb, var(--color-correct) 80%, black)` hoặc hardcode `#1a5c2e` với comment
- `rgba(217, 119, 87, 0.3)` → `rgba(var(--color-primary-rgb), 0.3)` hoặc thêm token `--color-primary-glow`
- `#ede8df`, `#c4a882`, `#3d3530` trong codeBlock — **giữ nguyên** (CLI block không đụng)

**Files:** `src/components/QuizScreen.module.css`, `src/screens/HomeScreen.module.css`

---

### 9. Xóa dead CSS — `QuizScreen.module.css`

**Files:** `src/screens/HomeScreen.module.css`, `src/components/QuizScreen.module.css`

### 10. Merge duplicate CLI utils — `src/utils/cliUtils.js`
- `splitIntoSegments`, `looksLikeCli`, `CLI_PROMPT_RE`, `PROSE_RE` tồn tại trong cả `QuizScreen.jsx` và `OptionCard.jsx`
- Extract ra `src/utils/cliUtils.js`

**Files:** `src/utils/cliUtils.js` (mới), `src/components/QuizScreen.jsx`, `src/components/OptionCard.jsx`

---

### 11. Fix easing curve — `QuizScreen.module.css`, `OptionCard.module.css`, `HomeScreen.module.css`

**Files:** tất cả `.module.css`

---

### 12. Fix multipleBanner position — `QuizScreen.jsx`

## Risks
- SVG icons cần size/stroke nhất quán — dùng `width=16 height=16 strokeWidth=1.5` cho tất cả
- `grid-template-rows` animation cho viBlock cần inner wrapper div — đừng quên thêm vào JSX
- Merge CLI utils: 2 file có regex hơi khác nhau — dùng version trong `OptionCard.jsx` (mới hơn, có `CLI_KEYWORD_RE`)
